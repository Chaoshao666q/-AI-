# 初中智慧课堂 AI 学习助手

基于生成式 AI 的初中人教版（六三制）智慧课堂教学辅助系统

---

## 📖 项目背景与简介

本系统是一款面向初中教育阶段的智慧教学辅助工具，旨在结合生成式人工智能大模型的能力，为一线教师提供高效的课堂互动支持、智能测评工具与数据化学情分析手段。系统覆盖九大学科，采用触摸屏友好界面设计，适配教室多媒体大屏环境，助力实现‘因材施教’与‘精准教学’。

---

## ✨ 核心功能特性 (v1.5)

### 🤖 AI 全科答疑辅导
- **多学科启发式教学**：覆盖初中九大学科，针对不同学科（如语文、数学、英语等）定制专属的启发式问答逻辑。先梳理核心概念，再引导解题思路，避免直接给出答案。
- **智能输入过滤**：自动识别并过滤无意义的乱发信息，引导学生提出具体有效的学习问题。
- **记录与回溯**：自动归档师生互动记录，支持按学科、时间维度筛选及导出，便于教学复盘。

### 📝 智能组卷与随堂测评
- **精细化组卷配置**：支持学科、年级、单元、课时、题量、难度梯度及题型（选择/填空/简答）的灵活配置。
- **考点聚焦出题**：可基于教材章节考点库，精准限定考点进行出题，生成高质量原创题目。
- **多题型智能批阅**：
    - **客观题**：自动比对答案。
    - **主观题（填空/简答）**：AI 自动批阅，简答题支持按点给分并提供详细评语。
- **激励性反馈**：批阅后生成个性化鼓励评语，并支持语音朗读评语。
- **数据沉淀**：全量保存答题数据，为学情分析提供支撑。

### 📚 结构化全科知识库
- **目录式浏览**：严格按照教材目录结构（科目→年级→单元→课时）梳理呈现知识点。
- **AI 一键生成**：支持 AI 一键生成、刷新或补充章节知识点，内容自动清理 Markdown 格式，保持纯净文本显示。
- **全文检索**：内置检索引擎，支持关键词快速定位相关内容。
- **记忆与续读**：记忆‘最后阅读位置’，支持‘继续学习’模式，无缝衔接上一次浏览。

### 📊 数据驱动的学情分析
- **可视化报表**：可视化展示班级整体在各章节的正确率分布与掌握情况。
- **多维度分析**：提供考点错误率排行、题型得分率对比、知识点掌握度画像。
- **典型错题归集**：自动归集高频错题并进行错误归因分析。
- **AI 教学建议**：基于足量答题数据（≥5题），AI 生成针对性的教学改进建议与个性化学习路径。

### 📋 学习动机测评 (MSLQ)
- **专业量表**：内置 18 题完整版学习动机量表，支持‘前测-后测’实验场景。
- **折叠式面板**：测评记录侧边栏支持展开/折叠，优化大屏操作体验。
- **数据管理**：支持历史测评记录的查询、统计与导出。

### ⚙️ 系统设置与模型管理
- **多模型兼容**：默认集成智谱AI (GLM) 和阿里云百炼 (通义千问) 免费模型。同时兼容所有 OpenAI API 标准接口，支持自定义添加模型。
- **用量精细管控**：提供模型 Token 用量统计面板，支持周期额度（每日/每月/一次性）管理，额度不足时自动提醒。
- **数据安全**：支持数据库一键备份与恢复，以及章节/考点、学情报告、错题本的标准化导入与导出。
- **双模智能语音**：
    - **离线模式**：基于 Windows SAPI，无网络延迟，响应迅速。
    - **在线模式**：基于 Edge-TTS，音质更佳，支持多种音色切换。网络不佳时自动降级为离线模式。
- **触摸与适配优化**：全局支持触摸屏滑动，UI 字体大小可调节，并针对 Windows 7 环境做了专门的 Emoji 兼容处理。

## 🚀 环境要求与运行方式

**最低配置要求**：
- 操作系统：Windows 7 / 10 / 11（64 位）
- 硬件配置：内存 2GB 以上（推荐 4GB 及以上）

**使用方式**：
1. **一键运行版（推荐一线教师使用）**：
   - 下载 `初中智慧课堂AI学习助手_v1.5.exe` 至本地电脑。
   - 双击运行，程序将在同级目录自动释放数据库文件 `smart_classroom_data.xlsx`（内含初中全科目教材结构化数据）。
   - 首次使用请前往【系统设置】配置 AI 模型 API Key。
2. **源码运行版（适合技术人员调试）**：
   - 安装 Python 依赖库：`pip install requests openpyxl edge-tts pygame pywin32`
   - 运行主程序：`python main.py`

## ⚙️ 系统配置与使用指南

1. **API Key 配置**：首次使用需在【系统设置】中配置有效的大模型 API Key（推荐智谱AI GLM-4-Flash 或阿里云百炼通义千问）。
2. **自定义模型接入**：在【模型管理】中可添加任何兼容 OpenAI API 的第三方模型，并设置额度周期以控制成本。
3. **数据管理**：
   - 支持教材章节/考点、学情分析报告、错题本的 Excel 导入与导出。
   - 提供数据库文件（`smart_classroom.db`）的一键备份与恢复功能。
4. **个性化设置**：可根据实际教学需求调整教材版本、地区、班级名称、UI 字体大小及语音模式。

## ❓ 常见问题解答（FAQ）

**Q**: 启动后提示‘请先配置 AI API Key’，应如何处理？
**A**: 请进入软件【系统设置】→【AI 功能配置】模块，正确填写 API Key，选择对应模型并点击‘测试连接’。若提示额度不足，可切换至其他已配置的模型或添加新的模型。

**Q**: 学情分析模块没有数据或无法生成建议？
**A**: 学情分析与 AI 教学建议依赖一定量的答题数据。请确保学生已通过【随堂答题】模块完成至少 5 道题目，系统将自动触发数据分析与建议生成。

**Q**: 如何自定义修改教材章节或考点内容？
**A**: 有两种方式：1. 直接编辑程序目录下的 `smart_classroom_data.xlsx` 文件中的‘章节’和‘考点’工作表；2. 通过【系统设置】中的‘数据导入’功能，导入符合模板格式的自定义 Excel 数据。

**Q**: 软件是否支持多班级学情数据管理？
**A**: 支持。在【系统设置】中切换‘当前班级名称’，不同班级的答题记录、学情报告与错题本将自动隔离存储。

**Q**: 在线语音朗读没有声音或很慢？
**A**: 在线语音依赖网络，首次合成可能需要几秒钟。如果网络不佳，程序会自动降级为离线语音。您也可以在设置中手动切换为“离线语音”以获得即时响应。
