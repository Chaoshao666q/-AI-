初中智慧课堂 AI 学习助手
安装与使用说明


一、作品基本信息

项目	信息
作品名称	初中智慧课堂 AI 学习助手
适配版本	人教版六三制初中全学科


二、运行环境
本程序支持 Windows 7/10/11 系统，分为可运行版和源码版两种使用方式：
1. 可运行版（推荐）
•无需安装 Python 环境，解压即可直接运行
•系统要求： Windows 7/10/11 系统（Windows 7运行兼容版），剩余磁盘空间≥200MB
•数据库 yishui_suyang2026_scc_data.db
•AI配置文件 ai_config.json
2. 源码版
•需要安装 Python 3.8 及以上版本
•需要安装依赖库：执行 pip install requests pyttsx3


三、安装与运行步骤
可运行版使用步骤：
1.解压 第二实验中学_AI学习助手_可运行程序.zip 压缩包到任意文件夹
2.双击文件夹中的 初中智慧课堂AI学习助手.exe 即可启动程序
3.Windows 7运行兼容版
源码版使用步骤：
1.解压源码压缩包
2.打开命令提示符，进入源码目录
3.执行 pip install requests pyttsx3 安装依赖
4.执行 第二实验中学_AI学习助手_源码.py 启动程序


四、功能使用指南
1. AI 智能答疑
•选择你的年级和学科
•在输入框输入你要问的问题，点击「发送问题」
•AI 会给出针对性的解答，你还可以点击「朗读」按钮让 AI 朗读回答
2. 随堂智能测试
•选择年级、学科、章节、课时，设置题目难度、题量和题型
•点击「生成测试题」，AI 会为你生成针对性的练习题
•输入你的答案，点击「提交答案」，AI 会自动为你判题，给出解析
•完成所有题目后，会自动统计你的正确率
3. 学习记录
•你所有的问答记录和答题记录都会自动保存在本地
•你可以按学科筛选查看历史记录，也可以清空所有记录
4. 系统设置
•第一次使用需要配置 AI 的 API Key，你可以在通义千问的官网获取
•配置完成后可以点击「测试连接」验证配置是否有效
•离线语音功能默认已就绪，无需网络就可以朗读文本


五、注意事项
1.AI 功能需要网络连接，离线状态下只能使用本地记录和语音功能
2.请不要泄露你的 API Key，避免被盗用
3.所有的学习记录都保存在本地电脑，不会上传到网络
4.如果程序无法启动，请尝试右键以管理员身份运行
